This is dmljava project folder for Eclipse (3.4.0+)

First need to have svn support in Eclipse:
Help->Software Updates...->Available Software->Add Site
http://www.polarion.org/projects/subversive/download/eclipse/2.0/ganymede-site/
OK, expand, expand Subversive SVN Connectors
select SVNKit 1.2.0 Implementation (Optional)
Install, restart eclipse when prompted
-------------

Window->Show View->Other->SVN->SVN Repositories
right click, New->Repository Location
URL: https://demlinks.svn.sourceforge.net/svnroot/demlinks/trunk/dmljava/
Finish
right click on the repository, Check Out

should check out a project named "dmljava"
